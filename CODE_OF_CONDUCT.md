# Code of Conduct

This project follows the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: we expect contributors and maintainers to interact respectfully and professionally. The full text — including the scope, our responsibilities, and the categories of conduct that are not acceptable — is at the link above. We adopt it as written, with one substitution: report violations to the channel in [`SECURITY.md`](./SECURITY.md) rather than the placeholder address in the Covenant template.

## Enforcement

The maintainers of this repository (members of the `metasession-dev` organisation with admin or maintain permissions) are responsible for enforcement. Reports are confidential. We follow the Covenant's [enforcement guidelines](https://www.contributor-covenant.org/version/2/1/code_of_conduct/#enforcement-guidelines) — Correction → Warning → Temporary Ban → Permanent Ban — proportional to the impact and pattern of the conduct.

## Scope

The Code applies in all project spaces: this repository's issues, pull requests, discussions, releases, and any community space (Slack, email lists, conferences) where someone is identifiably participating in the project.
